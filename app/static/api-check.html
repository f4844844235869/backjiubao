后端接口完整性检查页

这个页面用于手工串联当前后端接口，快速确认登录、当前用户、门店、组织、角色权限、员工档案这些主流程是否正常。

默认管理员通常是 admin@example.com / changethis。如果你改过 `.env`，请用你自己的账号密码。

B 端密码登录

POST /auth/backend/password-login

当前用户

GET /auth/me

检查当前用户、角色、权限、数据范围是否符合预期。

创建门店

POST /stores/
