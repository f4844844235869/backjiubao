## 当前结论

- 小程序侧现在也支持“当前门店上下文”概念
- 同手机号关联到多家门店员工时，不带门店上下文会返回全部关联员工
- 传入 `X-Current-Store-Id` 或 `current_store_id` 后，只返回当前门店下的关联员工
- 这能让小程序在多门店场景下明确区分“当前是从哪家门店来的请求”
