## 当前结论

- 小程序用户即使具备 `employee.read`，没有 `iam.user.read` 时仍不能访问普通用户列表
- 角色变更后，已有登录 token 的权限会立即按数据库最新状态收敛
- 同一员工可同时属于多个门店，并可按当前门店上下文访问对应门店的数据
